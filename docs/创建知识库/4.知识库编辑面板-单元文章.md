# 知识库编辑面板-单元文章

### 富文本功能

- 基本富文本功能
  - 文字加粗
  - 文字倾斜
  - 文字划线
  - 有序列表
  - 无序列表
  - 多级标题  
  - 改变字号
- 插入代码块
- 添加task
- 插入图片

![sss](https://file.songxiwen.com.cn/metagraph-doc/create-repository-page.png)

### 根据选中文字查找知识点

- 选中一段文本，点击查找，唤起查找知识点弹窗

### 引用（Mention）知识点

- 引用知识点，并绑定到单元
- 引用知识点，但不绑定到单元

### 解析文本 
- 支持Word文本解析(目前仅docx)
- 解析并渲染文本内容
- 提取文本关键词
- 将关键词创建为备选知识点
- 将解析出的文本创建为新单元
